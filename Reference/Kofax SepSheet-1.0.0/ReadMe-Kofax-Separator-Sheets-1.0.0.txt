Introduction
============
This package contains the Kofax Separator Sheet Utility.


Installation
============
To install the Separator Sheet Utility, do the following:

1. Make sure you have installed Microsoft.NET Framework 4.0 or later. Download it from the Microsoft website.

2. Extract the ZIP file to a network or local folder. It contains the files for running the utility.


Uninstallation
==============
Delete the folder where you extracted the ZIP file.

  
Starting the Kofax Separator Sheet Utility
==========================================
Run SepSheet.exe to run the utility. See the provided help for instructions.
